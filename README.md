# 🤖 Adima Bot - Multi-Service AI Telegram Assistant

Advanced Telegram bot with multiple AI service fallback and free voice synthesis.

## ✨ Features

- 🧠 **Multi-AI Support** - GROQ, Gemini, OpenRouter, OpenAI
- 🔄 **Automatic Fallback** - Never goes offline
- 🎤 **Natural Voices** - Sarvam AI, Edge TTS, Google TTS
- 🌍 **Multi-Language** - 9+ languages supported
- 📊 **Real-time Stats** - Monitor service health
- 💬 **Context Memory** - Remembers conversations
- 🆓 **100% Free** - Using free-tier APIs

## 🚀 Quick Start

### 1. Prerequisites

- Python 3.9 or higher
- Telegram account
- At least one AI API key (GROQ recommended - free)

### 2. Installation

```bash
# Clone repository
git clone https://github.com/asnzodiac/adima_bot.git
cd adima_bot

# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
